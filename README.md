# Task Management Web Application

A FastAPI-based web application for managing daily tasks with user authentication and task tracking.

## Features

- User Registration
- User Login
- Add Tasks
- Mark Tasks as Complete
- Delete Tasks
- Dashboard Statistics
- Responsive UI

## Tech Stack

- FastAPI
- SQLAlchemy
- SQLite
- HTML
- CSS
- Jinja2

## Installation

```bash
pip install -r requirements.txt
uvicorn app:app --reload
```
